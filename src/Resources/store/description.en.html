Shopmon Connector - Centralized Monitoring for Your Shopware Shops

FroshShopmon is the official connector plugin for Shopmon - the centralized monitoring dashboard for all your Shopware instances from FriendsOfShopware.

What is Shopmon?

Shopmon is a free, open-source monitoring platform that allows you to manage and monitor multiple Shopware shops from a single dashboard. Whether you're an agency managing dozens of client shops or a developer overseeing multiple environments - Shopmon gives you complete visibility without switching between admin panels.

What does this plugin do?

This plugin creates a secure integration in your Shopware shop with carefully selected, minimal permissions. It generates Base64-encoded credentials that you can easily copy and use to connect your shop to Shopmon.

Key Features

  • Automatic Integration Setup - No manual configuration needed. The plugin creates everything automatically.
  • Minimal Permissions - Only the necessary read permissions are granted for secure monitoring.
  • Easy Configuration - Simply copy the Base64-encoded credentials from the plugin settings.
  • FroshTools Support - If FroshTools is installed, Shopmon can perform health checks on your shop.
  • Cache Management - Clear your shop's cache directly from the Shopmon dashboard.

Shopmon Features

Once connected, Shopmon provides:

  • Real-time shop status monitoring
  • Version tracking for Shopware and all extensions
  • Security update notifications
  • Scheduled task overview and management
  • Queue monitoring
  • Performance monitoring with daily sitespeed checks
  • Email alerts for status changes
  • Multi-organization and team support

Permissions

The plugin creates an integration with the following read-only permissions:

  • Read access to apps and plugins
  • Read access to system configuration
  • Read access to scheduled tasks
  • Read access to FroshTools (if installed)
  • Cache clear and info permissions

Getting Started

  1. Install and activate the FroshShopmon plugin
  2. Navigate to Settings → Extensions → FroshShopmon in your admin panel
  3. Copy the Base64-encoded integration data
  4. Add your shop to Shopmon using the credentials

Open Source
