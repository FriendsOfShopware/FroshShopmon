Shopmon Connector - Zentrales Monitoring für deine Shopware Shops

FroshShopmon ist das offizielle Connector-Plugin für Shopmon - das zentrale Monitoring-Dashboard für alle deine Shopware-Instanzen von FriendsOfShopware.

Was ist Shopmon?

Shopmon ist eine kostenlose, Open-Source Monitoring-Plattform, mit der du mehrere Shopware-Shops über ein einziges Dashboard verwalten und überwachen kannst. Egal ob du eine Agentur bist, die dutzende Kunden-Shops betreut, oder ein Entwickler, der mehrere Umgebungen im Blick haben muss - Shopmon gibt dir volle Transparenz, ohne zwischen Admin-Panels wechseln zu müssen.

Was macht dieses Plugin?

Dieses Plugin erstellt eine sichere Integration in deinem Shopware-Shop mit sorgfältig ausgewählten, minimalen Berechtigungen. Es generiert Base64-kodierte Zugangsdaten, die du einfach kopieren und verwenden kannst, um deinen Shop mit Shopmon zu verbinden.

Hauptfunktionen

  • Automatisches Integrations-Setup - Keine manuelle Konfiguration nötig. Das Plugin erstellt alles automatisch.
  • Minimale Berechtigungen - Nur die notwendigen Lese-Berechtigungen werden für sicheres Monitoring vergeben.
  • Einfache Konfiguration - Kopiere einfach die Base64-kodierten Zugangsdaten aus den Plugin-Einstellungen.
  • FroshTools-Unterstützung - Wenn FroshTools installiert ist, kann Shopmon Health-Checks durchführen.
  • Cache-Verwaltung - Leere den Cache deines Shops direkt über das Shopmon-Dashboard.

Shopmon-Funktionen

Nach der Verbindung bietet Shopmon:

  • Echtzeit-Shop-Status-Monitoring
  • Versionsverfolgung für Shopware und alle Erweiterungen
  • Benachrichtigungen über Sicherheitsupdates
  • Übersicht und Verwaltung von Scheduled Tasks
  • Queue-Monitoring
  • Performance-Monitoring mit täglichen Sitespeed-Checks
  • E-Mail-Benachrichtigungen bei Statusänderungen
  • Multi-Organisation und Team-Unterstützung

Berechtigungen

Das Plugin erstellt eine Integration mit folgenden Nur-Lese-Berechtigungen:

  • Lesezugriff auf Apps und Plugins
  • Lesezugriff auf Systemkonfiguration
  • Lesezugriff auf Scheduled Tasks
  • Lesezugriff auf FroshTools (falls installiert)
  • Cache-Lösch- und Info-Berechtigungen

Erste Schritte

  1. Installiere und aktiviere das FroshShopmon-Plugin
  2. Navigiere zu Einstellungen → Erweiterungen → FroshShopmon in deinem Admin-Panel
  3. Kopiere die Base64-kodierten Integrationsdaten
  4. Füge deinen Shop bei Shopmon mit den Zugangsdaten hinzu

Open Source

Sowohl dieses Plugin als auch Shopmon sind Open Source und werden von FriendsOfShopware entwickelt. Beiträge sind willkommen auf GitHub.
